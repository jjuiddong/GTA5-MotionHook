# GTA5-MotionHook
GTA5 Motion Data Hooking using ScriptHookV SDK with VS2015 C++
- ScriptHookV_SDK_1.0.617.1a
- Motion Data
    - in NativeSpeedo project
    - send UDP, 127.0.0.1:10001
    - float heading
    - float roll
    - float pitch
    - float velocity_x
    - float velocity_y
    - float velocity_z
